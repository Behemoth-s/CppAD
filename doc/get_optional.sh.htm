Download and Install The CppAD Optional Packages

Syntax
bin/get_optional.sh

Purpose
If you are using Unix, this command will download and install all of the optional packages that can be used with CppAD.

Distribution Directory
This command must be executed in the distribution directory .

prefix
The prefix for installing the optional packages is 

prefix="$HOME/prefix/cppad"
It can be changed by editing the line above in the file bin/get_optional.sh

Output
The output for each of these scripts is suppressed. You can get the output for an particular script running it directly from the command line.
