Cppadcg Speed: Sparse Jacobian

Syntax
flag = sparse_jacobian_c(
    subgraph, optimize, seed, size, nnz, x, y
)

Prototype 

extern "C" int sparse_jacobian_c(
    int           subgraph   ,
    int           optimized  ,
    int           seed       ,
    int           size       ,
    int           nnz        ,
    const double* x          ,
    double*       y
);

Purpose
Computes a sparse Jacobian corresponding to the function sparse_jac_fun .

subgraph
If it is one (zero), the subgraph_jac_rev method is used (is not used) to compute the sparse Jacobian. This function was only implemented for one value of subgraph . You can create a new version of this function by calling cppadcg_sparse_jacobian_cg.cpp .

optimize
If it is one (zero), the optimized (non-optimized) version of the computation is used. This function was only implemented for one value of optimize .

seed
is the random number seed used to choose the row and column vectors. This function was only implemented for one value of seed .

size
Is the dimension of the argument space for the function. This function is only implemented few values of size .

nnz
is the number of non-zeros in the sparsity pattern for the Jacobian.

x
This is a vector of length size containing the point at which to evaluate the derivative.

y
Is a vector with size nnz containing the non-zeros in the sparse Jacobian. The input value of it's elements does not matter. Upon return, for k = 0 , ..., nnz-1 . the k-th component of y is the k-th non-zero in the sparse Jacobian.

flag
If the value of subgraph , optimize , and seed are the same, and size was one of the sizes, when the current version of sparse_jacobian_c was generated, flag = 0 . If the value of subgraph , optimize , or seed is different, flag = 1 . If the value of subgraph , optimize , and seed are the same and size is not one of the sizes, flag = 2 .
