Cppadcg Speed: Gradient of Determinant by Minor Expansion

Syntax
flag = det_minor_grad_c(optimize, size, x, y)

Prototype 


int det_minor_grad_c(int optimized, int size, const double* x, double* y)

Purpose
Computes the gradient of the determinant for a square matrix.

optimize
If it is zero (one), the non-optimized (optimized) version of the computation is used.

size
Is the row and column dimension of the square matrix. This function is only implemented for one value of size . You can create a version of this function for a different value of size by calling cppadcg_det_minor.cpp .

x
Is the values in the matrix at which we are evaluating the gradient (stored in either row or column major order). The length of this vector is the product of its row and column dimension; i.e., size*size .

y
Is the gradient of the determinant function. It size is the same as x . The length of this vector is the same as x . The input value of it's elements does not matter. Upon return the j-th component of y is the partial of the determinant with respect to the j-th component of x .

flag
This value is zero if the value of size is passed in can be handled by this source code; see cppadcg_det_minor_cg.cpp . Otherwise, y is not used and the return value of flag is 1.
